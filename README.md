**Проект "Дополнение спецификации требованиий для приложения Chatty"**: https://docs.google.com/document/d/1nxSXkSBerEj6b_L5weltFrXzsooRn4PcFu2H6XqLCvg/edit?usp=sharing

В данном проекте необходимо было завершить спецификацию требований к ПО для второго релиза Chatty (мобильное приложение для обучения английскому языку). 
В задачи входило:
- дополнить спецификацию функциональными требованиями для функции карточек для запоминания новых слов.
- подготовить требования к новой функции просмотра видео:
   1) обновить диаграмму UML Use Case;
   2) добавить новые табличные варианты использования;
   3) написать функциональные требования для функции видео.
- для новой функции просмотра видео необходимо заполнить другие разделы спецификации:
  1) атрибуты качества;
  2) бизнес-правила;
  3) глоссарий.

_Использованный стек: FR, UML Use Case, Use Case._
